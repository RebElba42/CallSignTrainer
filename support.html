Hilfe & Anleitung

CallSign Trainer ist eine Lern-App, mit der du Amateurfunkrufzeichen als Morsecode (CW) direkt im Browser üben kannst – mobil und am Desktop. Die App ist kostenlos, Open Source und speichert deine Einstellungen lokal.

Bedienung:
Wähle die gewünschte Sprache oben rechts. Starte das Quiz mit Start, beende mit Stop. Im Automatikmodus läuft das Quiz automatisch weiter, im manuellen Modus kannst du jede Aufgabe selbst weiterklicken.

Geschwindigkeit (WPM)
Stellt die Abspielgeschwindigkeit des Morsecodes in Worten pro Minute ein.
Pause (Sekunden)
Legt die Pause zwischen den Aufgaben fest.
Wiederholen
Wie oft das aktuelle Rufzeichen wiederholt wird.
Farnsworth-WPM
Erlaubt langsame Abstände bei schneller Zeichen-Geschwindigkeit (Farnsworth-Methode).
Tonhöhe (min/max Hz)
Stellt die minimale und maximale Tonhöhe für die Morsezeichen ein.
Test Morse
Spielt ein Test-Rufzeichen zur Probe ab.
Verzögerung vor Ansage
Wartezeit vor der Voransage (z.B. "VVV" oder Sprache).
Verzögerung vor Morse
Wartezeit vor dem Start des Morsecodes.
Verzögerung vor Lösung
Wartezeit, bevor die Lösung angezeigt wird.

Störgeräusch-Typ
Wähle zwischen Weißes Rauschen, Pinkes Rauschen, Braunes Rauschen oder QRN (natürliche Störungen).
Störgeräusch-Level
Stellt die Lautstärke des Störgeräuschs ein.
QRM-Level
Simuliert Störungen durch andere Stationen (QRM).
QSB-Level
Simuliert Schwund (QSB) – das Signal wird lauter und leiser.
Test Morse (mit Störung)
Spielt ein Test-Rufzeichen mit Störgeräuschen ab.

Voransage
Wähle, ob vor jedem Rufzeichen "VVV" in Morse oder eine Sprachansage kommt.
Rufzeichen-Kategorie
Wähle, ob internationale oder nur europäische Rufzeichen trainiert werden.
Design/Thema
Wechsle das Farbschema der App (Darkmode, verschiedene Bootstrap-Themes).
Stimme
Wähle die Stimme für Sprachansagen aus.
Stimme testen
Spielt eine Testansage mit der gewählten Stimme ab.
Einstellungen zurücksetzen
Setzt alle Einstellungen auf Standardwerte zurück.

Help & Guide

CallSign Trainer is a learning app to practice amateur radio callsigns as Morse code (CW) directly in your browser – on mobile and desktop. The app is free, open source, and stores your settings locally.

Usage:
Choose your language at the top right. Start the quiz with Start, stop with Stop Button. In automatic mode the quiz continues automatically, in manual mode you can proceed to the next task yourself.

Speed (WPM)
Sets the playback speed of Morse code in words per minute.
Pause (seconds)
Sets the pause between tasks.
Repeat
How often the current callsign is repeated.
Farnsworth-WPM
Allows slow spacing with fast character speed (Farnsworth method).
Tone (min/max Hz)
Sets the minimum and maximum tone frequency for Morse code.
Test Morse
Plays a test callsign for trial.
Delay before announcement
Wait time before the pre-announcement ("VVV" or speech).
Delay before Morse
Wait time before Morse code starts.
Delay before solution
Wait time before the solution is shown.

Noise type
Choose between white noise, pink noise, brown noise or QRN (natural noise).
Noise level
Sets the volume of the noise.
QRM level
Simulates interference from other stations (QRM).
QSB level
Simulates fading (QSB) – the signal gets louder and softer.
Test Morse (with noise)
Plays a test callsign with noise.

Pre-announcement
Choose whether "VVV" in Morse or a speech announcement is played before each callsign.
Callsign category
Choose whether to train international or only European callsigns.
Design/Theme
Switch the app's color scheme (dark mode, various Bootstrap themes).
Voice
Select the voice for speech announcements.
Test voice
Plays a test announcement with the selected voice.
Reset settings
Resets all settings to default values.
